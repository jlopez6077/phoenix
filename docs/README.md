## 🦅 Phoenix Basic Library

Phoenix Basic Library is a collection of simple, reusable hardware modules developed as part of a learning journey in SystemVerilog, Cocotb, Verilator, Icarus Verilog, and GTKWave.
The goal of this project is to build a foundational set of FPGA components — such as FIFOs, RAMs, and UART interfaces — from first principles, without relying on vendor IP.
This project is also a way to developed a workflow specific to fpga development that includes designing, verification, and documentation
